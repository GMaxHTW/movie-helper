# Default Vue3 Project Setup for a Single Page Aplication

## Specifications
- Vue3
- VueCLI
- WebHistory Router
- css sass preprocessor
- gsap.library
- FontAwesomeIcons
- CMS Rest Client (Api)
- Server .htaccess config

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
